When you're programming, there are times when you want to give a certain command to the computer more than one time. Maybe you want to print out the value of each element of an array -- remember that an array has a list of items -- or, if you're writing some code to do graphics, maybe you want to tell something on the page to move in a certain direction multiple times. Obviously you don't want to type the command over and over again. That's boring, wastes screen space, and you might get tired and not type the command the number of times you wanted to.

What if there were an easy way to tell the computer to do a command a certain number of times?

As a matter of fact, there is! Programmers call this important concept iteration. You'll need to use a mix of what you remember from both giving computers commands and conditionals to use iteration. Conditionals are useful because we often want to tell the computer to do something until a conditional statement is false. For example, if we've created a quiz on addition and we're asking the user what 2 + 3 is, we might want to direct the computer to print out "Try again!" until the input is equal to 5.

There are two main kinds of iteration. The while loop is the easier one to understand, and it looks like this:

while statement
action

This says that while the statement isn't true, the computer should perform the action. In the addition quiz example used above, where we want the user to put in the correct answer for 2 + 3, your loop might look something like this:

while input doesn't equal 5
print "Try again!"

This puts together all the stuff we've learned. From the conditionals lesson, we get that this iteration, or loop, will keep happening until the input equals 5. From the commands lesson, we get that the print statement tells the computer to write something to your screen. And from the variable types lesson, we know that "Try again!" is a string.

There is also a for loop. We've included one for loop exercise in case you want to try it, but it's a little more complex than the while loop--don't feel too bad if it confuses you!

Here's the general for loop syntax: for initial valuestatementwhat to change every time the loop runs
action

So a for loop actually starts out with three parts! The first part sets some variable values to start the loop with. The second part is a conditional statement; just like in the while loop, the loop will repeat until this statement is false. And the third part contains changes which should happen at the end of each time the loop runs. If you're still confused, check out what printing out the numbers from 1 to 10 might look like:

for i = 1, i < 11, add 1 to i
print i

Look closely at this loop. We start the loop by making i equal to 1, and we're only going to stop when i isn't less than 11. So this will print out the numbers from 1 to 10 because the computer will first print i, then increase the value of i by 1, and then print out i if the new value is still less than 11. The loop will stop itself once it sees that i is more than 11. Otherwise, we would just print out numbers forever!
